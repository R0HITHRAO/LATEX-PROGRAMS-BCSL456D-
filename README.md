# LATEX-PROGRAMS-BCSL456D
LaTeX Programs (BCSL456D)

This repository contains the complete set of LaTeX programs and documents developed as part of the **BCSL456D** laboratory course. The goal of this repository is to demonstrate proficiency in typesetting technical documents, mathematical equations, academic papers, and presentations using LaTeX.

## 🚀 Overview

LaTeX is a high-quality typesetting system widely used for medium-to-large technical or scientific documents. This repository serves as a structured collection of lab exercises, templates, and assignments required for the course curriculum.

## 🛠️ Prerequisites & Installation

To run and compile the `.tex` files locally, you will need a LaTeX distribution and an editor.

### 1. LaTeX Distributions
*   **Windows:** [MiKTeX](https://miktex.org/)
*   **macOS:** [MacTeX](https://www.tug.org/mactex/)
*   **Linux:** TeX Live (`sudo apt install texlive-full`)

### 2. Recommended Editors
*   [VS Code](https://code.visualstudio.com/) (with the *LaTeX Workshop* extension)
*   [TeXstudio](https://www.texstudio.org/)
*   [Overleaf](https://www.overleaf.com/) (Cloud-based, no installation required)
